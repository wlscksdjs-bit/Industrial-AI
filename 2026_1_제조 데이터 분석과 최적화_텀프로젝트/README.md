# ⚙️ 제조 데이터 분석과 최적화 텀프로젝트 (Term Project)

**Condition-Based Maintenance (CBM) using BiLSTM and XAI**

충북대학교 산업인공지능학과 대학원 1학년 1학기(2026학년도 1학기) '제조 데이터 분석과 최적화' 과목의 기말 텀프로젝트 산출물 아카이브입니다.

---

## 🎯 프로젝트 개요

본 프로젝트는 제조업 스마트팩토리 공정 내 중요 설비의 고장 및 이상 상태를 선제적으로 감지하는 **상태 기반 정비 (CBM, Condition-Based Maintenance)** 파이프라인의 신뢰성 향상을 목적으로 합니다. 
시계열 설비 센서 데이터를 효과적으로 처리하기 위해 **양방향 LSTM (BiLSTM, Bidirectional LSTM)** 신경망 모델을 설계 및 훈련하고, 인공지능 예측 모델의 결정을 현업 전문가가 해석할 수 있도록 **설명 가능한 인공지능 (XAI, eXplainable AI)** 기법을 접목했습니다.

이를 통해 고장을 단순 감지하는 것을 넘어, "어떤 센서 인자가 이상 작동에 결정적인 기여를 했는지" 실시간 기여도를 시각화함으로써 예지 보전 의사결정의 신뢰도를 대폭 강화했습니다.

## 💻 핵심 소스코드 구조 및 모델 개요
소스코드는 [2026254019_진찬언/텀프로젝트.py](./2026254019_진찬언/텀프로젝트.py) 경로에 저장되어 있습니다.
* **데이터 전처리**: 다중 센서 시계열 피처 마트 생성, 정규화, 결측치 보간 및 슬라이딩 윈도우 기반 3D Tensor 형태 변환.
* **BiLSTM 신경망**: Forward, Backward 양방향에서 맥락 정보를 추출해 시계열 특성을 포착하는 아키텍처 구현.
* **XAI 모듈**: 대표적인 XAI 프레임워크를 연동하여 중요도 점수(Attribution Score) 산출 및 센서별 영향도 그래프 시각화.
