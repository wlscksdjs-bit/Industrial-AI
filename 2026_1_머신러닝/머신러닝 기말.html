머신러닝 시험 문제 및 해설

[문제 1] 혼동행렬(Confusion Matrix)

문제: 혼동행렬을 2x2 표로 작성하고, TP, FP, TN, FN을 사용하여 성능지표를 아는 대로 설명하시오.

1. 혼동행렬 (2x2 표)

예측 (Predicted Class)
Positive (양성 예측) Negative (음성 예측)
실제
(Actual Class)		 Positive
(실제 양성)		 TP (True Positive)
(실제 양성을 양성으로 맞춤) 		FN (False Negative)
(실제 양성을 음성으로 틀림)
Negative
(실제 음성)		 FP (False Positive)
(실제 음성을 양성으로 틀림) 		TN (True Negative)
(실제 음성을 음성으로 맞춤)

2. 성능 지표 설명

  • 정확도 (Accuracy): 전체 예측 데이터 중 올바르게 예측한 비율.
    $Accuracy = \frac{TP + TN}{TP + FP + FN + TN}$
  • 정밀도 (Precision): 모델이 '양성(Positive)'으로 예측한 것 중 실제 '양성'인 비율. (FP를 줄이는 것이 중요할 때 사용, 예: 스팸 메일 분류)
    $Precision = \frac{TP}{TP + FP}$
  • 재현율 (Recall / Sensitivity / True Positive Rate): 실제 '양성(Positive)'인 것 중 모델이 '양성'으로 올바르게 예측한 비율. (FN을 줄이는 것이 중요할 때 사용, 예: 암 진단)
    $Recall = \frac{TP}{TP + FN}$
  • F1-Score: 정밀도와 재현율의 조화평균. 데이터 클래스 비율이 불균형할 때 모델의 성능을 정확하게 평가하기 위해 사용.
    $F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}$
  • 특이도 (Specificity / True Negative Rate): 실제 '음성'인 것 중 모델이 '음성'으로 올바르게 예측한 비율.
    $Specificity = \frac{TN}{TN + FP}$

[문제 2] 선형회귀 모델 평가 (SSE)

문제: 선형회귀 모델 평가를 위해 4개의 데이터포인트 실제 정답 $y = [3, -0.5, 2, 7]$와 모델 예측 $\hat{y} = [2.5, 0.0, 2, 8]$가 주어졌다. 오차제곱합(SSE)을 구하는 과정과 결과값을 구하시오.
(참고: 문제에서 '평균제곱오차(SSE)'라고 명시되었으나, 기호 SSE는 오차제곱합을 의미하므로 SSE를 기준으로 계산합니다.)

1. SSE (Sum of Squared Errors, 오차제곱합) 계산식

$$SSE = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

2. 계산 과정

각 데이터 포인트에 대해 (실제값 - 예측값)의 제곱을 구합니다.

  • 1번째 데이터: $(3 - 2.5)^2 = (0.5)^2 = 0.25$
  • 2번째 데이터: $(-0.5 - 0.0)^2 = (-0.5)^2 = 0.25$
  • 3번째 데이터: $(2 - 2)^2 = (0)^2 = 0$
  • 4번째 데이터: $(7 - 8)^2 = (-1)^2 = 1$

3. 결과값

$$SSE = 0.25 + 0.25 + 0 + 1 = \mathbf{1.5}$$

※ 참고: 만약 평균제곱오차(MSE, Mean Squared Error)를 구해야 한다면, SSE를 데이터 개수인 4로 나누어 $1.5 / 4 = 0.375$ 가 됩니다.

[문제 3] 시그모이드 함수와 임계치

문제: 시그모이드 함수를 작성하고, 입력이 0일 때 이진 분류에서 임계치(Threshold) 관점에서의 물리적 의미를 쓰시오.

1. 시그모이드 함수 (Sigmoid Function)

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

2. 입력이 0일 때의 물리적 의미

입력 $x = 0$을 함수에 대입하면 $\sigma(0) = \frac{1}{1 + e^0} = \frac{1}{1 + 1} = 0.5$ 가 됩니다.

이진 분류(Binary Classification)에서 출력값 0.5는 분류를 결정짓는 기준선인 임계치(Decision Boundary) 역할을 합니다. 즉, 모델이 양성(Class 1)과 음성(Class 0)일 확률을 정확히 50:50으로 판단하여 가장 불확실성이 높은 지점이며, 입력값이 0보다 커지면 양성으로, 0보다 작아지면 음성으로 분류가 전환되는 물리적인 분기점(전환점)을 의미합니다.

[문제 4] KNN에서의 K 선택과 모델 성능

문제: KNN(K-Nearest Neighbor)에서 임의값 K를 선택할 때, K를 1로 설정할 때와 K를 큰 값으로 설정할 때 과적합(Overfitting), 과소적합(Underfitting)을 넣어 설명하시오.
  • $K=1$로 설정할 때 (과적합 - Overfitting):
    가장 가까운 이웃 단 1개의 데이터만 보고 클래스를 결정합니다. 이는 학습 데이터의 노이즈(이상치)까지 모두 학습하게 되어 결정 경계(Decision Boundary)가 매우 복잡해집니다. 학습 데이터에는 완벽하게 동작하지만 새로운 데이터에 대한 예측력은 떨어지는 과적합(Overfitting) 상태가 됩니다. (High Variance)

  • $K$를 매우 큰 값으로 설정할 때 (과소적합 - Underfitting):
    주변의 수많은 데이터를 고려하므로 결정 경계가 지나치게 단순해지고 부드러워집니다. 만약 K가 전체 데이터 개수와 같다면 무조건 가장 데이터가 많은 클래스로만 예측하게 됩니다. 이는 데이터의 세부적인 지역적 패턴을 무시하게 되어 학습 데이터조차 제대로 맞추지 못하는 과소적합(Underfitting) 상태가 됩니다. (High Bias)

[문제 5] SVM 마진(Margin) 산식

문제: SVM(서포트 벡터 머신)에서 놈(Norm) $||w||^2$를 활용해서 마진을 표현하는 산식을 쓰시오.

SVM에서 초평면(Hyperplane)의 방정식이 $w^T x + b = 0$일 때, 양성 클래스와 음성 클래스를 가르는 서포트 벡터 사이의 총 거리인 마진(Margin)의 기하학적 너비는 다음과 같이 표현됩니다.

$$Margin = \frac{2}{||w||}$$

SVM은 이 마진을 최대화(Maximize)하는 것이 목적입니다. $\frac{2}{||w||}$를 최대화하는 것은 수학적으로 $\frac{1}{2} ||w||^2$를 최소화(Minimize)하는 최적화 문제로 변환하여 풉니다. 따라서 $||w||^2$를 활용한 목적 함수(Objective Function) 산식은 다음과 같습니다.

$$\min \frac{1}{2} ||w||^2$$

[문제 6] 엔트로피와 지니 계수

문제: 의사결정나무(Decision Tree)에서 불순도를 측정하는 엔트로피 함수, 지니 함수 수학적 계산식을 작성하시오.

(※ $c$는 클래스의 개수, $p_i$는 해당 노드에서 $i$번째 클래스가 차지하는 비율(확률)을 의미합니다.)

1. 엔트로피 (Entropy) 함수

$$Entropy(S) = -\sum_{i=1}^{c} p_i \log_2(p_i)$$

2. 지니 불순도 (Gini Impurity) 함수

$$Gini(S) = 1 - \sum_{i=1}^{c} (p_i)^2$$

[문제 7] PCA와 LDA (지도/비지도 학습)

문제: PCA, LDA에 대한 설명 및 목적, 그리고 어느 것이 지도학습인지 쓰시오.
  • PCA (Principal Component Analysis, 주성분 분석)
    • 설명 및 목적: 데이터의 분산(Variance)을 최대한 보존하는 새로운 직교 축(주성분)을 찾아 고차원의 데이터를 저차원으로 축소하는 기법입니다. 데이터의 전반적인 구조를 파악하고 노이즈를 제거하는 데 목적이 있습니다.
    • 학습 유형: 정답(Label) 데이터 없이 특징량만 사용하므로 비지도 학습(Unsupervised Learning)입니다.

  • LDA (Linear Discriminant Analysis, 선형 판별 분석)
    • 설명 및 목적: 클래스 간 분산(Between-class scatter)은 최대화하고, 클래스 내부 분산(Within-class scatter)은 최소화하는 축을 찾아 데이터를 투영시킵니다. 즉, 데이터가 어떤 클래스에 속하는지 가장 잘 분류(구분)할 수 있는 차원으로 축소하는 것이 목적입니다.
    • 학습 유형: 차원 축소를 위해 데이터의 정답(Class Label) 정보를 활용해야 하므로 지도 학습(Supervised Learning)입니다.

[문제 8] 군집화 모델 평가 (MSE와 엘보우 포인트)

문제: K-Means 클러스터링 등에서 쓰이는 MSE와 엘보우 포인트(Elbow Point) 개념을 쓰시오.
  • MSE (Mean Squared Error, 평균제곱오차 / 응집도):
    군집화에서 MSE는 각 데이터 포인트와 자신이 속한 군집의 중심점(Centroid) 간의 거리의 제곱의 평균을 의미합니다. (오차제곱합인 SSE를 전체 데이터 개수로 나눈 값과 같습니다.) MSE 값이 작을수록 데이터들이 군집의 중심에 촘촘하게 모여 있음(군집이 잘 형성됨)을 뜻합니다.

  • 엘보우 포인트 (Elbow Point):
    군집의 개수($K$)를 늘려가며 MSE(또는 SSE)를 계산하여 그래프로 그렸을 때, 오차값이 급격하게 감소하다가 감소 폭이 완만해지며 꺾이는 지점(마치 팔꿈치 모양)을 말합니다. 이 지점의 $K$ 값이 군집화의 성능과 계산 비용을 타협하는 최적의 군집 개수(Optimal K)로 간주됩니다.

[문제 9] 파라미터와 하이퍼파라미터

문제: 파라미터(Parameter)와 하이퍼파라미터(Hyperparameter)의 개념을 설명하고 예시를 각 하나씩 쓰시오.
  • 파라미터 (Parameter - 매개변수)
    • 개념: 모델 내부에 존재하며, 학습 데이터로부터 모델이 스스로 학습하여 업데이트하고 찾아내는 값입니다. 사용자가 직접 수동으로 설정할 수 없습니다.
    • 예시: 인공신경망의 가중치(Weight, $W$) 및 편향(Bias, $b$), 선형 회귀의 회귀 계수.

  • 하이퍼파라미터 (Hyperparameter - 초매개변수)
    • 개념: 모델 외부에 존재하며, 모델을 학습시키기 전에 사용자가 직접 경험이나 휴리스틱에 의해 수동으로 설정해 주어야 하는 값입니다. 모델의 학습 과정이나 구조를 제어합니다.
    • 예시: KNN의 $K$ 개수, 학습률(Learning Rate), 에포크(Epoch) 수, 신경망의 은닉층(Hidden Layer) 개수.

[문제 10] 앙상블 학습 (Ensemble)

문제: 앙상블의 보팅(Voting) 다수결 결과값 개념과, Major voting(Hard voting) 계산법, Weight voting(Soft/Weighted voting) 계산법을 쓰시오.

10-1. 보팅 (Voting) 및 다수결 결과값 개념

보팅은 서로 다른 알고리즘을 가진 여러 개의 머신러닝 모델(분류기)을 결합하여 같은 데이터에 대해 예측하게 한 뒤, 각 모델의 예측 결과를 투표(Voting)를 통해 최종 예측 결과로 선정하는 방식입니다. 단일 모델보다 일반화 성능을 높이는 데 목적이 있습니다.

10-2. 계산 방법

  • Major Voting (Hard Voting - 다수결 투표):
    • 계산법: 각 모델이 예측한 결과(클래스 레이블) 중 가장 많이 득표한 클래스(최빈값)를 최종 결과로 선정합니다.
    • (예: 모델 A→클래스1, 모델 B→클래스1, 모델 C→클래스2 예측 시, 다수결에 의해 '클래스 1'로 최종 결정)

  • Weight Voting (가중치 투표 - Weighted/Soft Voting 유사 개념):
    • 계산법: 각 모델의 성능이나 신뢰도에 따라 가중치(Weight)를 부여합니다. 각 클래스별로 모델들의 투표수에 가중치를 곱하여 합산한 뒤, 가장 높은 가중치 합을 얻은 클래스를 최종 결과로 선정합니다.
    • (예: 성능이 좋은 모델A 가중치 0.6, 모델B 0.2, 모델C 0.2 일 때, 모델 A가 클래스2, B와 C가 클래스1을 예측하면. 클래스1 점수 = $0.2+0.2=0.4$, 클래스2 점수 = $0.6$. 따라서 '클래스 2'로 최종 결정)

※ 참고: 실무에서는 모델들이 예측한 클래스별 '확률값'을 평균 내는 Soft Voting 방식을 가장 널리 사용합니다.
